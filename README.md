# Charan Reddy — AI Engineer Portfolio

Professional React portfolio with a glassmorphism visual style, animated AI orbit component, project cards, technical stack, experience timeline, and responsive layout.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite, usually:

```bash
http://localhost:5173
```

## Build for Production

```bash
npm run build
npm run preview
```

## Main Files

- `src/main.jsx` — React portfolio layout and components
- `src/styles.css` — glassy background, texture, animations, responsive UI
- `src/data/portfolio.js` — resume/project data

## Customize

Edit `src/data/portfolio.js` to update contact details, projects, skills, and experience.
